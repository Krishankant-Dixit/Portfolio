# Resume Placeholder

Please place your resume PDF file here and name it `resume.pdf`.

The resume download feature in the portfolio will link to this file.

If you don't have a PDF ready, you can:
1. Create one using tools like Canva, Adobe, or Google Docs
2. Export it as PDF
3. Name it `resume.pdf`
4. Place it in this `public` folder

The download buttons in the Hero and Contact sections will automatically work once the file is added.
